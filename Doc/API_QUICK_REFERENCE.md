# XFramework 全功能 API 速查手册 (Exhaustive Quick Reference)

> 本文档为 XFramework 的进阶 API 指南，详细列出了框架中所有核心工具类、扩展方法及其典型用法。

## 1. 场景与对象查询 (Query & Finder)

### 1.1 `UObjectFinder`
极致性能的场景对象提取工具。
- `Find<T>(string name)`: 沿场景根节点查找。
- `Find(string name)`: 返回 GameObject。
- `FindRecursive<T>(Transform parent, string name)`: 在父节点下递归查找。

### 1.2 `Extend.Unity` (Transform 增强)
- `transform.FindRecursive(string name)`: 递归查询子物体。
- `transform.GetChildren()`: 获取直接子物体。
- `transform.SetX/Y/Z(float val, VectorType type)`: 链式修改坐标/旋转/缩放。
- `transform.ResetLocal()`: 重置 Local Position, Rotation (Identity), Scale (One)。

## 2. 向量与数学运算 (Vector & Math)

### 2.1 向量扩展 (`Vector2`, `Vector3`)
- `vec.WithX/Y/Z(float)`: 修改分量并返回新向量。
- `vec.AddX/Y/Z(float)`: 在分量基础上累加并返回。
- `vec.XY()` / `XZ()` / `YZ()`: 跨维度转换。
- `vec.AlmostZero()`: 浮点误差安全的零值检测。
- `vec.Abs()`: 分量绝对值。
- `vec.Rotate(float degrees)`: 二维向量绕原点旋转。

### 2.2 `UUtility.Bezier` (贝塞尔曲线)
- `GetPoint(p0, p1, p2, p3, t)`: 返回三阶贝塞尔曲线上的一点。
- `GetTangent(p0, p1, p2, p3, t)`: 获取切线方向。

### 2.3 `UUtility.Grid` (网格运算)
- `WorldToGrid(pos, cellSize)`: 世界坐标转离散网格坐标。
- `GridToWorld(index, cellSize)`: 网格索引转世界坐标。

## 3. 动态模型生成 (UUtility.Model)
> 处理基于代码的 Mesh 构建，适用于 Procedural Generation。

### 3.1 基础形状生成器
所有方法返回 `UnityEngine.Mesh`，接受对应的 `Description` 结构体：
- `GenerateCubeMesh(CubeDescription)`: 支持圆角(Chamfer)的立方体。
- `GenerateSphereMesh(SphereDescription)`: 高精度合拢球。
- `GenerateCylinderMesh(CylinderDescription)`: 圆柱/棱柱。
- `GenerateCapsuleMesh(CapsuleDescription)`: 胶囊体。
- `GenerateTorusMesh(CircularRingDescription)`: 圆环。
- `GenerateArrowMesh(ArrowDescription)`: 3D 箭头。

### 3.2 进阶拓扑
- `GenerateTubeMesh`: 中空管状体。
- `GenerateStarMesh`: 平面/立体多角星。
- `GenerateHelixMesh`: 螺旋弹簧结构。
- `GenerateTetrahedron / Octahedron`: 正多面体。

## 4. 底层抽象工具 (Utility)

### 4.1 反射增强 `Utility.Reflection` (极度推荐)
- `CreateInstance<T>(params object[] args)`: 带优化的动态实例化。
- `GetSonTypes<T>(string assembly)`: 检索指定程序集内所有子类。
- `GetAssignableTypes(Type baseType)`: 获取所有继承或实现该类型的 Class。
- `MethodWrapperAction(target, methodInfo)`: 将反射得到的 `MethodInfo` 包装为高性能的 `Action/Func` 委托。
- `GetFields / GetMethods (filter)`: 带筛选器的成员获取。

### 4.2 文件与 IO `Utility.IO`
- `GetFileMD5(string path)`: 快速哈希校验。
- `ClearDirectory(string path)`: 清空目录但不删除根文件夹。
- `SearchFiles(path, pattern)`: 深度递归文件搜索。

### 4.3 文本解析 `Utility.Text`
- `SplitPathName(path)`: 智能拆分文件路径与扩展名。
- `GetAfterStr(org, value)`: 截取指定字符后的子串。
